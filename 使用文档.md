# 大挖故障追踪系统使用文档

## 1. 系统概述

大挖故障追踪系统是面向大挖公司内部员工的实时故障看板系统，旨在实现故障记录的可视化、可追踪、可协作、可分析，提升故障响应与闭环效率。

### 1.1 系统特点

- **实时监控**：实时展示当日故障情况，包括故障数量、优先级分布等
- **历史查询**：支持多维度筛选和查询历史故障记录
- **趋势分析**：通过图表直观展示故障趋势和分布
- **故障管理**：支持新增、编辑、删除故障记录，以及批量导入
- **个人待办**：展示当前用户的待办故障和处理进度
- **AI助手**：提供智能问答功能，帮助分析故障数据
- **响应式设计**：适配PC端和移动端

### 1.2 系统架构

- **前端技术**：Vue 3 + Vite + Element Plus + ECharts
- **路由管理**：Vue Router
- **状态管理**：Vue 3 Composition API
- **数据存储**：localStorage（模拟）

## 2. 系统功能

### 2.1 登录功能

- **飞书授权登录**：点击登录按钮模拟飞书授权登录流程
- **自动跳转**：登录成功后自动跳转到当日故障页面
- **退出登录**：点击顶部导航栏中的退出登录按钮

### 2.2 当日故障

- **故障统计**：展示总故障数、紧急故障、重要故障、一般故障和未闭环故障的数量
- **故障卡片**：按优先级分类展示当日故障，支持点击查看详情
- **故障详情**：点击故障卡片可查看详细信息，包括故障描述、原因分析、对策方案等
- **刷新数据**：点击刷新按钮可更新数据
- **快速导航**：点击添加按钮可快速跳转到故障管理页面

### 2.3 历史故障

- **多维度筛选**：支持按早会日期、分类、机型、机号、生产日期、服务工程师、是否对策、风险评估、是否闭环、整机生产地、故障部位等维度筛选
- **保存查询条件**：可保存常用查询条件，方便下次快速使用
- **故障列表**：展示故障的详细信息，支持排序
- **故障详情**：点击查看按钮可查看故障详细信息
- **数据导出**：支持导出故障数据为CSV格式

### 2.4 故障趋势

- **时间范围筛选**：支持按月份范围筛选数据
- **多维度筛选**：支持按机型、机号、故障部位、分类等维度筛选
- **统计卡片**：展示总故障数、紧急故障、重要故障、一般故障和平均故障率
- **图表展示**：
  - 故障数量趋势图：按月份展示故障数量变化
  - 故障类型分布图：展示不同类型故障的占比
  - 故障部位分布图：展示不同故障部位的占比
  - 故障率趋势图：按月份展示故障率变化
- **数据导出**：支持导出故障趋势数据为CSV格式

### 2.5 故障管理

- **新增故障**：点击新增按钮添加新的故障记录
- **编辑故障**：点击编辑按钮修改已有故障记录
- **删除故障**：管理员可点击删除按钮删除故障记录
- **更新进展**：点击更新进展按钮更新故障处理进度
- **批量导入**：支持通过Excel文件批量导入故障记录
- **状态切换**：通过开关直接切换故障的闭环状态

### 2.6 个人待办

- **待办统计**：展示总待办、紧急待办、重要待办和已完成待办的数量
- **分类查看**：支持查看全部、待处理和已完成的待办
- **待办详情**：点击查看按钮可查看待办故障的详细信息
- **更新进展**：点击更新按钮可更新待办故障的处理进度

### 2.7 AI助手

- **智能问答**：可询问系统记录中故障率最高的故障部位、故障率最高的机型等
- **自动交互**：用户一定时间不点击，AI助手会自动与用户交互
- **半隐藏功能**：双击AI助手可使其半隐藏在网页右边

## 3. 操作指南

### 3.1 登录系统

1. 打开系统登录页面
2. 点击「飞书授权登录」按钮
3. 系统模拟登录成功后自动跳转到当日故障页面

### 3.2 查看当日故障

1. 在侧边栏点击「当日故障」菜单
2. 查看故障统计卡片，了解故障概览
3. 点击故障卡片查看详细信息
4. 点击刷新按钮更新数据
5. 点击添加按钮跳转到故障管理页面

### 3.3 查询历史故障

1. 在侧边栏点击「历史故障」菜单
2. 设置筛选条件，如日期范围、分类、机型等
3. 点击「查询」按钮执行筛选
4. 点击「保存条件」按钮保存当前筛选条件
5. 点击「重置」按钮清空筛选条件
6. 点击常用查询条件标签快速加载保存的筛选条件
7. 点击故障列表中的「查看」按钮查看详细信息
8. 点击导出按钮导出故障数据

### 3.4 分析故障趋势

1. 在侧边栏点击「故障趋势」菜单
2. 设置时间范围和其他筛选条件
3. 点击「查询」按钮更新图表
4. 点击「重置」按钮清空筛选条件
5. 查看统计卡片和图表，分析故障趋势
6. 点击导出按钮导出趋势数据

### 3.5 管理故障

1. 在侧边栏点击「故障管理」菜单
2. 点击「新增」按钮添加新故障
3. 填写故障表单，点击「确定」保存
4. 点击故障列表中的「编辑」按钮修改故障信息
5. 点击「更新进展」按钮更新故障处理进度
6. 点击「删除」按钮删除故障记录（仅管理员）
7. 点击「批量导入」按钮导入故障数据

### 3.6 处理个人待办

1. 在侧边栏点击「个人待办」菜单
2. 查看待办统计卡片，了解待办概览
3. 点击标签页切换查看全部、待处理和已完成的待办
4. 点击「查看」按钮查看待办详情
5. 点击「更新」按钮更新待办处理进度

### 3.7 使用AI助手

1. 点击右下角的AI助手图标
2. 在对话框中输入问题，如「最近一个月故障率最高的是哪个机型」
3. 点击「发送」按钮或按Enter键提交问题
4. 查看AI助手的回答
5. 双击AI助手图标使其半隐藏

## 4. 注意事项

### 4.1 数据安全

- 系统使用localStorage存储数据，仅用于演示，实际部署时应使用后端数据库
- 敏感信息如用户密码等不应存储在前端

### 4.2 浏览器兼容性

- 推荐使用Chrome、Firefox、Edge等现代浏览器
- 不支持IE浏览器

### 4.3 性能优化

- 对于大量数据的查询，建议使用分页和筛选条件减少数据量
- 图表加载可能需要一定时间，特别是在数据量较大时

### 4.4 权限管理

- 系统模拟了基本的权限管理，如删除功能仅对管理员开放
- 实际部署时应实现完整的权限管理系统

### 4.5 移动端适配

- 系统支持移动端访问，但部分功能可能在小屏幕设备上体验不佳
- 建议在PC端使用以获得最佳体验

## 5. 常见问题

### 5.1 登录失败

**问题**：点击登录按钮后无反应
**解决方案**：检查网络连接，确保系统正常运行

### 5.2 数据不更新

**问题**：点击刷新按钮后数据未更新
**解决方案**：系统使用模拟数据，刷新不会改变数据内容

### 5.3 导出功能无法使用

**问题**：点击导出按钮后无反应
**解决方案**：检查浏览器是否阻止了下载，尝试使用Chrome浏览器

### 5.4 AI助手不响应

**问题**：AI助手对话框不显示或无响应
**解决方案**：刷新页面，确保浏览器支持JavaScript

### 5.5 移动端布局错乱

**问题**：在手机上访问时布局错乱
**解决方案**：尝试横屏查看，或使用PC端访问
